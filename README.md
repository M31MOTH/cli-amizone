## Command line program for Amizone (Amity University's Intranet Zone)

**Project is WIP.**

## Requirements
* Python3
* Install the following dependencies using pip
    * ```$ sudo pip3 install beautifulsoup4 requests```

## Usage
```$ python Amizone.py```

## What works
* Student login
* Student `My Courses`
    * Course code, Course name, type, attendance
